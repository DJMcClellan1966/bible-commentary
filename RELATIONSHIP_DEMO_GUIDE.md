### On Your iPad/Phone:
1. Make sure your device is on the same Wi-Fi network
2. Note the IP address shown when you start the server (e.g., `192.168.1.49`)
3. Open Safari (or any browser)
4. Go to: `http://YOUR_IP:8000` (e.g., `http://192.168.1.49:8000`)

## Troubleshooting

### Port Already in Use
- The script automatically finds a free port
- Check the console output for the correct port number

### Can't Access on iPad
- Make sure both devices are on the same Wi-Fi network
- Check the IP address shown in the console
- Try disabling firewall temporarily

### No Results Showing
- Make sure you entered text in the input fields
- Check the browser console for errors
- Try refreshing the page
