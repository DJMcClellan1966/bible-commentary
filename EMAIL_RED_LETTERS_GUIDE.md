# How to Email the Red Letters Book

## Quick Steps

### Option 1: Email EPUB File (Best for iPad) ‚≠ê

1. **Find the file:**
   - Location: `red_letters_book_formats/red_letters_book.epub`
   - Size: ~30-50 KB (small, easy to email)

2. **Attach to email:**
   - Open your email client (Gmail, Outlook, etc.)
   - Create new email
   - Attach the file: `red_letters_book.epub`
   - Send to yourself (or recipient)

3. **On iPad:**
   - Open email on iPad
   - Tap the EPUB attachment
   - Choose "Open in Books"
   - Book appears in your library!

### Option 2: Email HTML File (Works Anywhere)

1. **Find the file:**
   - Location: `red_letters_book_formats/red_letters_book.html`
   - Size: ~30-50 KB

2. **Attach to email:**
   - Attach `red_letters_book.html` to email
   - Send to yourself

3. **On iPad/Device:**
   - Open email
   - Tap HTML file
   - Opens in browser
   - Can add to Home Screen

## Step-by-Step: Windows Email

### Using Gmail (Web):

1. **Open Gmail** in your browser
2. **Click "Compose"**
3. **Click the paperclip icon** (Attach files)
4. **Navigate to:** `red_letters_book_formats/red_letters_book.epub`
5. **Select the file** and click "Open"
6. **Enter your email address** (or recipient's)
7. **Subject:** "Red Letters Book - For iPad"
8. **Click "Send"**

### Using Outlook:

1. **Open Outlook**
2. **Click "New Email"**
3. **Click "Attach File"**
4. **Browse to:** `red_letters_book_formats/`
5. **Select:** `red_letters_book.epub`
6. **Enter email address**
7. **Click "Send"**

### Using Windows Mail:

1. **Open Mail app**
2. **Click "New Mail"**
3. **Click "Attach"** (paperclip icon)
4. **Select:** `red_letters_book.epub`
5. **Enter recipient**
6. **Send**

## Troubleshooting

### File won't attach:
- Check file isn't open in another program
- Try right-click ‚Üí Send to ‚Üí Mail recipient (Windows)
- Use drag-and-drop method

### Can't find file:
- Navigate to: `red_letters_book_formats` folder
- Look for `red_letters_book.epub`

### Email too large:
- These files are tiny (~30-50 KB)
- Should email without issues
- If problem, try HTML version

### Won't open on iPad:
- Make sure it's EPUB format (not HTML)
- Try opening in Files app first, then share to Books
- Check file extension is `.epub`

## Quick Command (Windows)

You can also use this PowerShell command to email:

```powershell
# Open email with attachment (if default mail client is set)
Start-Process "mailto:your-email@example.com?subject=Red Letters Book&body=Attached is the Red Letters book in EPUB format."
```

Then manually attach the file.
