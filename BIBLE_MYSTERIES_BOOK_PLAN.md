## How to Generate

Run the book generator:
```bash
python generate_bible_mysteries_book.py
```

This will:
1. Load Bible versions
2. Generate each chapter using AI
3. Ground all content in actual Scripture
4. Create cross-references
5. Save to `bible_mysteries_book/` directory

## Output Files

- `bible_mysteries_book.md` - Complete book
- `chapter_*.md` - Individual chapters
- `book_metadata.json` - Book structure and metadata

## Note

The book generation uses grounded AI generation, ensuring all content is based on actual Bible verses and their connections throughout Scripture. Each chapter explores how the mystery relates to the whole of the Bible.
