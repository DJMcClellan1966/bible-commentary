Bible Commentary & Study App
=============================

QUICK START:
1. Double-click "BibleCommentaryApp.exe"
2. Wait for the browser to open automatically
3. Start using the app!

FEATURES:
- Bible Commentary Generator
- AI-Powered Search
- Study Plans and Notes
- Bible Character Models
- Quantum AI Features
- And much more!

TROUBLESHOOTING:
- If browser doesn't open, go to: http://127.0.0.1:8000
- If port is in use, close other instances
- First run may take a moment to extract files

SUPPORT:
- Check the app interface for help
- Visit API docs at: http://127.0.0.1:8000/docs

Enjoy studying the Bible!
